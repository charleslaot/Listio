# LISTIO

## SUMMARY

Listio is a web application that allows the user to create, edit and populate playlists fetching song's metadata from the Spotify API.

## LIVE DEMO

[Listio Demo](https://nameless-springs-69015.herokuapp.com/)

## TECHNOLOGIES USED

- HTML
- CSS
- JavaScript
- JQuery
- Font Awesome
- Google Fonts
- Spotify API
- NodeJS
- ExpressJS
- MongoDB
- Mongoose
- JWT Authentication
- Mocha
- Chai
- TravisCI
- Heroku
- JSDoc
